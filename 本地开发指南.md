# 本地开发指南

## 前提条件
✅ 已安装 Node.js（访问 https://nodejs.org/ 下载 LTS 版本）

## 启动步骤

### 1. 启动后端服务器

在项目根目录打开终端：

```powershell
cd c:\Users\42047\Downloads\code\strings\server
npm install
npm start
```

后端将运行在：http://localhost:3000

---

### 2. 启动前端开发服务器

**打开新的终端窗口**（不要关闭后端终端），运行：

```powershell
cd c:\Users\42047\Downloads\code\strings\client
npm install
npm run dev
```

前端开发服务器将运行在：http://localhost:5173

---

## 访问应用

打开浏览器访问：**http://localhost:5173**

- ✅ 前端自动刷新（修改代码即时生效）
- ✅ API 请求自动代理到后端
- ✅ 支持热模块替换（HMR）

---

## 常见问题

### 端口被占用
如果提示端口被占用：
- 后端：修改 `server/` 目录下的端口配置
- 前端：Vite 会自动尝试其他端口

### 后端连接失败
确保后端服务器在 3000 端口运行（检查第一个终端窗口）

---

## 停止服务

在各自的终端窗口按 `Ctrl + C` 停止服务
